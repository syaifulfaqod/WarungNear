# Product Overview

WarungNear Frontend adalah aplikasi web berbasis React + Vite yang menjadi antarmuka utama untuk pelanggan dan pemilik toko.

Frontend ini berfungsi sebagai penghubung antara pengguna dengan sistem backend Node.js melalui REST API.

Sistem frontend memiliki dua area utama:

1. Customer Website
2. Owner Dashboard POS

Customer dapat mencari barang berdasarkan lokasi terdekat, melihat stok, harga, dan informasi toko.

Owner dapat mengakses dashboard POS untuk mengelola produk, stok, dan transaksi.
